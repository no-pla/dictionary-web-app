# Dictionary Web App

사전 API를 활용하여 제작한 사전 웹 앱.

## feature

1. 사전 단어 검색 기능 (api/dictionary)
- API route를 사용하여 서버 측에서 API 핸들링 할 수 있도록 처리
2. 다크 모드
3. 폰트 변경
4. 데스크탑/태블릿/모바일 적응형 디자인
